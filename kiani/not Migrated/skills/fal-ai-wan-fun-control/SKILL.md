# Wan 2.2 Fun Control

Generate pose or depth controlled video using Alibaba-PAI's Wan 2.2 Fun

**Endpoint:** `fal-ai/wan-fun-control`
**Source:** https://fal.ai/models/fal-ai/wan-fun-control/api

---

## Quick Start

### 1. Install the Client

```bash
npm install --save @fal-ai/client
```

### 2. Set Your API Key

```bash
export FAL_KEY="YOUR_API_KEY"
```

### 3. Submit a Request

```javascript
import { fal } from "@fal-ai/client";

const result = await fal.subscribe("fal-ai/wan-fun-control", {
  input: {
        "prompt": "your value here",
        "control_video_url": "https://example.com/input.png"
    },
  logs: true,
  onQueueUpdate: (update) => {
    if (update.status === "IN_PROGRESS") {
      update.logs.map((l) => l.message).forEach(console.log);
    }
  },
});
console.log(result.data);
console.log(result.requestId);
```

## Authentication

Set the `FAL_KEY` environment variable, or configure in code:

```javascript
import { fal } from "@fal-ai/client";
fal.config({ credentials: "YOUR_FAL_KEY" });
```

> **Important:** When running client-side, use a server-side proxy to protect your API key.

## Input Schema

| Parameter | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `prompt` | string | **Yes** |  | The prompt to generate the video. |
| `shift` | float | No | `5` | The shift for the scheduler. |
| `preprocess_video` | boolean | No | `false` | Whether to preprocess the video. If True, the video will be preprocessed to depth or pose. |
| `reference_image_url` | string | No |  | The URL of the reference image to use as a reference for the video generation. |
| `fps` | integer | No | `16` | The fps to generate. Only used when match_input_fps is False. |
| `match_input_num_frames` | boolean | No | `true` | Whether to match the number of frames in the input video. |
| `guidance_scale` | float | No | `6` | The guidance scale. |
| `preprocess_type` | enum: `depth`, `pose` | No | `"depth"` | The type of preprocess to apply to the video. Only used when preprocess_video is True. |
| `control_video_url` | string | **Yes** |  | The URL of the control video to use as a reference for the video generation. |
| `negative_prompt` | string | No | `""` | The negative prompt to generate the video. |
| `num_frames` | integer | No | `81` | The number of frames to generate. Only used when match_input_num_frames is False. |
| `seed` | integer | No |  | The seed for the random number generator. |
| `num_inference_steps` | integer | No | `27` | The number of inference steps. |
| `match_input_fps` | boolean | No | `true` | Whether to match the fps in the input video. |

## Queue API (Long-Running Requests)

### Submit to Queue

```javascript
const { request_id } = await fal.queue.submit("fal-ai/wan-fun-control", {
  input: {
        "prompt": "your value here",
        "control_video_url": "https://example.com/input.png"
    },
  webhookUrl: "https://optional.webhook.url/for/results",
});
```

### Check Status

```javascript
const status = await fal.queue.status("fal-ai/wan-fun-control", {
  requestId: "<request_id>",
  logs: true,
});
```

### Get Result

```javascript
const result = await fal.queue.result("fal-ai/wan-fun-control", {
  requestId: "<request_id>",
});
console.log(result.data);
```

## Tips

- Use `fal.subscribe` for quick scripts; use queue API for production workloads.
- Set `webhookUrl` on queue submit to get notified when processing completes.
- File inputs accept URLs, Base64 data URIs, or uploaded files via `fal.storage.upload(file)`.
